# 📊 Global Superstore Business Performance Dashboard

An interactive **Business Performance Dashboard** built in **Microsoft Excel** using **Pivot Tables**, **Pivot Charts**, **Excel Formulas**, and **Slicers**. This dashboard provides insights into sales performance, profitability, customer behavior, regional trends, shipment modes, and product performance using the **Global Superstore dataset** from Kaggle.

---

## 📌 Project Overview

This project demonstrates how Microsoft Excel can be used as a powerful Business Intelligence (BI) tool to transform raw sales data into meaningful and interactive business insights.

The dashboard allows users to analyze business performance across multiple dimensions such as:

* Sales
* Profit
* Orders
* Product Categories
* Sub-Categories
* Customers
* Regions
* Shipment Modes
* Order Priority

Interactive slicers enable users to filter the dashboard by **Year** and **Customer Segment**, making data exploration simple and dynamic.

## 🧮 Excel Features Used

### Pivot Tables

Used to summarize and aggregate large datasets efficiently.

Examples:

* Sales by Region
* Sales by Category
* Profit by Sub-Category
* Top Customers
* Shipment Mode Distribution

---

### Pivot Charts

Created interactive visualizations directly connected to Pivot Tables.

Chart types used include:

* Column Charts
* Bar Charts
* Doughnut Chart
* Combo Charts (Bar + Line)

---

### Excel Formulas

The dashboard uses several Excel formulas including:

* `SUM()`
* `COUNT()`
* `AVERAGE()`
* `IF()`
* `GETPIVOTDATA()`
* `DIVIDE (Profit / Sales)` for Profit Margin
* Calculated fields for KPI metrics

---

### Slicers

Interactive slicers were added for:

* Year
* Customer Segment

These allow users to filter all Pivot Tables and Pivot Charts simultaneously.
